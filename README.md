<h1 align="center">Prime Number Checker</h1>
A simple Python tool to quickly check whether a number is prime.

## Installation:
- Install Python
```sh
pkg install python
```
- Install Git
```sh
pkg install git
```

## How to run
```sh
git clone https://github.com/ShahriarAbidbd/PrimeNumberChecker
cd PrimeNumberChecker
python main.py
```
